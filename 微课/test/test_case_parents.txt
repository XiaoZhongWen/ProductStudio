角色:家长
1. 绑定学号
	1. 我的页面有绑定学号菜单 done
	2. 在绑定学号页面, 不输入学号, 提示请输入学号 done
	3. 在正确输入学号后, 点击添加按钮, 能正确绑定学员 done
	4. 当输入的学号不足9位数时, 提示学号必须是9位数 done
	5. 当输入的学号错误时, 提示绑定的学号不存在 done
