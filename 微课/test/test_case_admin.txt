		7.2.4 课程被绑定后, 显示课程的总体进度 done

				7.2.6.2 课程的进度信息需要被同步更新 
	7.3 添加课程
		7.3.1 必须填写课程类型、课程名称、课程时长后才能添加课程, 否则给予提示 done
		7.3.2 能够根据填写的课程信息成功添加课程 done

10. 班级
	10.1 添加班级按钮显示条件
		10.1.1 会员未过期 done
		10.1.2 从我的页面进入 done
	10.2 添加班级
		10.2.1 从添加机构页面进入, 添加班级页面显示相应机构 done
		10.2.2 从班级列表页面点击添加按钮进入, 如果路由时携带了机构id, 则同9.2.1; 如果没有携带机构id, 则添加班级页面显示该用户所有机构 done
		10.2.3 从班级列表点击班级卡片进入, 如果用户是班级所在机构的创建者, 则显示更新班级; 否则显示班级详情 done
		10.2.4 添加班级卡片
				10.2.4.1 填写班级名称 done
				10.2.4.2 若机构的课程或老师只有一个, 则自动选择数量只有一个的课程或老师 done
				10.2.4.3 若机构的课程或老师有多个, 则需要用户主动选择 done
				10.2.4.4 当选择课程后, 授课老师下拉框的数据变更为班级所在机构中教该课程的老师 done
				10.2.4.5 当选择老师后, 选择课程下拉列表数据变更为班级所在机构中该老师所教的课程 done
				10.2.4.6 卡片头部显示机构信息 done
				10.2.4.7 卡片中显示已经添加到机构的班级信息 done
				10.2.4.8 已经添加到机构的班级信息要考虑班级名称过长导致显示异常的问题 done
				10.2.4.9 卡片中显示班级的编辑表单 done
				10.2.4.10 班级的编辑表单中, 如果用户不是班级所属机构的创建者, 则班级名称、选择课程、选择老师、班级描述不能编辑, 班级学员栏目不能变更; 否则可以编辑, 可以变更班级学员栏目 done
				10.2.4.11 点击邀请按钮, 底部弹出选择班级学员视图; 视图中只显示该机构中绑定同一课程、同一老师且未被邀请的学员 done
				10.2.4.12 邀请完成后, 被邀请的学员显示在班级学员栏目中 done
				10.2.4.13 点击移除按钮, 底部弹出选择学员视图; 视图中只显示被邀请的班级学员 done
				10.2.4.14 移除完成后, 班级学员栏目的已邀请学员数据同步更新 done
				10.2.4.15 必须填写班级名称后, 才能添加; 否则给予提示 done
				10.2.4.16 点击已经添加的班级后, 如果用户是班级所属机构的创建者, 则编辑表单显示相应班级信息并能够被编辑, 同时添加按钮改为更新按钮 done
				10.2.4.17 点击添加或更新按钮能够正常添加或更新班级 done
