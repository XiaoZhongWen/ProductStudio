6. 课程
	6.1 添加课程按钮显示条件
		会员未过期 done
	6.2 课程列表
		6.2.1 显示自己匿名机构的课程以及所属机构的课程 done
		6.2.2 当所属机构为学员绑定课程并指定自己为授课老师时, 下拉刷新后能同步课程信息 done
		6.2.3 课程被绑定后, 显示课程的总体进度 done
		6.2.4 课程信息变更后, 课程列表中相应课程信息也要同步更新
			  6.2.4.1 在添加课程页面, 选择已添加的课程后, 可以变更课程图标、类型、名称、时长、描述 done
			  6.2.4.2 课程的进度信息需要被同步更新 
	6.3 添加课程
		6.3.1 必须填写课程类型、课程名称、课程时长后才能添加课程, 否则给予提示 done
		6.3.2 能够根据填写的课程信息成功添加课程 done
		6.3.3 点击已添加课程图标后, 显示该课程信息并能够编辑课程信息, 点击更新按钮后能够正确更新 done
		6.3.4 课程类型为“次课”时, 隐藏选择课程时间菜单 done
	6.4 删除课程
		6.4.1 长按课程图标, 显示删除课程按钮 done
		6.4.2 删除课程按钮显示时, 再次点击课程图标, 隐藏删除课程按钮 done
		6.4.3 如果不存在“在读”状态的学生选择了该课程, 则可以正常删除 done
		6.4.4 如果存在“在读”状态的学生选择了该课程, 则不可以删除并给提示
		6.4.5 课程被删除后, 课程列表同步更新 done

7. 学员

		会员未过期 done
	7.2 学员列表
		7.2.1 显示自己所有的学员(自己匿名机构的学员、所属机构中绑定课程时指定自己为授课老师的学员) done
		7.2.2 学员被添加或删除后, 学员列表信息同步更新 done
		7.2.3 点击课程图标进入学员的课程绑定页面 done
		7.2.4 点击日历进入学员的课程日历页面 done
		7.2.5 学员绑定课程后显示学员的课程进度 done
	7.3 添加学员
		7.3.1 学员的名字只能由家长变更 
		7.3.2 必须填写学员姓名、管理手机号后才能添加学员, 否则给予提示 done
		7.3.3 能够根据填写的学员信息成功添加学员 done
		7.3.4 添加学员的过程要显示进度 done
		7.3.5 点击学员头像进入学员的课程绑定页面 done
		7.3.6 只显示自己的匿名机构
	7.4 删除学员
		7.4.1 长按学员头像, 显示删除学员按钮 done
		7.4 2 删除学员按钮显示时, 再次点击学员头像, 隐藏删除学员按钮 done
		7.4.3 如果学员已经结课或未绑定课程, 可以正常删除 done
		7.4.4 如果学员未结课, 则不能删除并给提示 done

8. 绑定课程
	8.1 顶部视图显示学员基本信息, 如头像、昵称、学号、所属机构、课程进度 done
	8.2 要同步学员课程进度更新数据
	8.3 显示所有已绑定的课程信息 done
	8.4 绑定课程表单显示条件
		8.4.1 会员未过期 
		8.4.2 学员在匿名机构中还有未绑定的课程 done
	8.5 绑定课程表单
		8.5.1 必须先选择课程再选择老师, 否则老师下拉框无数据 done
		8.5.2 选择课程 -> 显示该学员在匿名机构中未绑定的课程 done
		8.5.3 选择老师 -> 显示老师信息 done
		8.5.4 选择课程后, 课程类型、课程所属机构名称、课程时长、课程描述自动填充 done
		8.5.5 填写总课时数、已耗课时数、课时单价及报名时间 done
		8.5.6 课程名称、授课老师、总课时数、已耗课时数、课时单价、报名时间全部填写完成后, 绑定课程按钮才可用 done
		8.5.7 绑定课程成功后, 获取授课老师以及绑定学员的用户的公众号openId集合, 若集合不为空, 则向各个openId发送课程绑定公众号订阅消息 done

9. 班级
	9.1 添加班级按钮显示条件
		9.1.1 会员未过期 done
		9.1.2 从我的页面进入 done
	9.2 班级列表
		9.2.1 显示自己匿名机构的班级以及所属机构中授课老师是自己的班级 done
		9.2.2 点击非自己创建的班级, 进入班级详情页面, 不能修改班级信息 done
		9.2.3 点击自己创建的班级, 进入更新班级页面, 可以修改班级信息 done
		9.2.4 同步添加|更新班级页面对班级数据的更新 done
	9.3 添加班级
		9.3.1 填写班级名称 done
		9.3.2 若课程或老师只有一个, 则自动选择数量只有一个的课程或老师 done
		9.3.3 若课程或老师有多个, 则需要用户主动选择 done
		9.3.4 授课老师下拉框的数据默认为自己 done
		9.3.5 选择课程下拉列表数据为自己创建的课程 done
		9.3.6 卡片头部显示匿名机构信息 done
		9.3.7 卡片中显示已经添加的班级信息 done
		9.3.8 已经添加的班级信息要考虑班级名称过长导致显示异常的问题 done
		9.3.9 卡片中显示班级的编辑表单 done
		9.3.10 班级的编辑表单中, 如果用户不是班级的创建者, 则班级名称、选择课程、选择老师、班级描述不能编辑, 班级学员栏目不能变更; 否则可以编辑, 可以变更班级学员栏目 done
		9.3.11 点击邀请按钮, 底部弹出选择班级学员视图; 视图中只显示该机构中绑定同一课程、同一老师且未被邀请的学员 done
		9.3.12 邀请完成后, 被邀请的学员显示在班级学员栏目中 done
		9.3.13 点击移除按钮, 底部弹出选择学员视图; 视图中只显示被邀请的班级学员 done
		9.3.14 移除完成后, 班级学员栏目的已邀请学员数据同步更新 done
		9.3.15 必须填写班级名称后, 才能添加; 否则给予提示 done
		9.3.16 点击已经添加的班级后, 如果用户是班级的创建者, 则编辑表单显示相应班级信息并能够被编辑, 同时添加按钮改为更新按钮 done
		9.3.17 点击添加或更新按钮能够正常添加或更新班级 done
