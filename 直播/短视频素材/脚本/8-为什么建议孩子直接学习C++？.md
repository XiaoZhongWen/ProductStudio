### 为什么建议孩子直接学习C++？

```
1. scratch不是生产工具，撇开不谈
2. 如果孩子未来想真正的走计算机编程这条路或者走科技特长，再或者未来想进入国家人工智能等高精尖领域，就一定要学C++；
举一个简单的例子，Python这个语言中使用的很多库，都是C++语言开发出来的；即C++开发出来给Python用，可以看到C++这个语言，它的一个重要性
3. 为什么走特长的孩子，尤其是初升高、高升大，都要学习C++,因为目前的信息学奥林匹克竞赛；小学、初中的叫CSP-J/S的认证，都是用C++去认证的，并没有用Python，因为这个语言，它能考察到孩子，是否具备这方面的特长素养，它对孩子的细心、严谨要求更高；它对孩子对编程语言的运用能力更高，对孩子的数学要求能力更高；
4. 当然Python和C++同时放在眼前，到底先学哪个，后学哪个；它没有明确的先后之分，而是跟个人的目标有关系，从我们教练的角度来讲，如果学习了C++，再返回来学习Python，基本上不需要老师教，自学就可以；但如果已经适应了Python再来学校C++，孩子会在心理产生一个障碍，就是C++会变的更难一些，它对语法等方面的要求更大一些；所以，如果是我选，如果孩子真的对这个有兴趣，并且有一定的时间，我也希望他通过这个去帮助到数学等各个学科；我会选C++；而且有一天，说不定，他可以走特长；Python学的再好，它也走不了特长，在咱们初升高里面，这就是原因
4. 虽然C++在产业界被公认为是非常难得编程语言，但是对于小孩子而言，咱们并不是让他们用C++去生产软件产品，而只是用C++表达算法逻辑，在这个意义上，孩子是不需要把C++学的很深入的，所以在学习难度上，会降低很多
```
